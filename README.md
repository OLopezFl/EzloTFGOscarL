## Instalación

Desde la raíz del proyecto:

1. composer install
2. npm install
3. Crear el Env a partir de .env.example
4. php artisan key:generate
5. Configurar variables DB_* en .env
6. php artisan migrate

## Arranque

En la Terminal 1:

1. composer run serve

En la terminal 2 Terminal 2:

1. npm run dev

Abrir en navegador:

- http://127.0.0.1:8000

o

1. composer run dev
